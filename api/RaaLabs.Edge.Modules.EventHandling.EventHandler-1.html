    Class EventHandler<T>

    An event handler for the given event type T. This is responsible for the "plumbing" between event producers and event consumers. It exposes a function "Produce(...)", which will be called by all event producers.

    On a new incoming event, the EventHandler class will iterate through all the consumers for its event type T, calling the "Handle(T @event)&quot;/&quot;HandleAsync(T @event)&quot; function of all these classes.

    For normal development, this class can be ignored by the developer.
